laboratório ICAD

## Alguns pontos importantes
- O webapp foi desenvolvido com o banco de dados sendo hosteado localmente, uma vez que as opçoes na nuvem do SQL server eram pagas.
    - O código para gerar o banco de dados com o qual estou trabalhando foi incluido em taskerrDbFinal.sql
    - É necessário atualizar a connectionString em appsettings.json para corresponder a novo server
    - Eu também optei por fazer um video mostrando a funcionalidade no meu computador, uma vez que podem ocorrer problemas na criação do novo banco de dados
        - https://www.youtube.com/watch?v=c8UPs28Gktc
